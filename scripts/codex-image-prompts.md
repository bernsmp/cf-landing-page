# Codex Image Prompts — The Remix Room

For Codex to generate 10 images using GPT-image-1 (or whichever GPT image model is current) in parallel with banana-squad's Gemini generation. Same 10 concepts, tuned for GPT's strengths. Compare the two sets, pick winners per card.

## Task for Codex

```
Generate 10 images for cognitivefingerprint.ai/remixroom. Use the latest GPT
image model. Save to:

/Users/maxb/Desktop/Vibe Projects/Cognitive Fingerprint Projects/cf-landing-page/public/remixroom-gpt/

Filename pattern: {id}.png (e.g. "05-naval-bourdain.png")

One image per prompt below. Aspect ratios specified. No text, no brand marks,
no celebrity likenesses (these are archetypal scenes, not portraits).

CF palette, apply to every image:
- Canvas: deep near-black #0a0a0b
- Primary: warm ivory #F0EDE6
- Accent: warm gold #ffb829 (used sparingly, ONE hit per image)

Quality bar: editorial cinematic photography, museum-lit still-life,
Dan Koe design universe, archival Criterion Collection feel. Not AI fantasy,
not 3D render, not illustration.

If any prompt triggers safety refusal, paraphrase once, skip specific names,
lean on archetypal objects and composition.
```
